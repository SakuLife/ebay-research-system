SerpApi（サープAPI）について
━━━━━━━━━━━━━━━━━━━━━━━━

■ SerpApiとは？
━━━━━━━━━━━━━━━━━━━━━━━━

Googleの検索結果をプログラムから取得できるサービスです。

通常、Googleで「商品名 価格」と検索すると、
Google Shoppingの結果として複数のショップの価格が表示されます。

SerpApiを使うと、この検索結果を自動で取得できるため、
「どのショップが最安値か」を一括で調べることができます。


■ なぜSerpApiを使うのか？
━━━━━━━━━━━━━━━━━━━━━━━━

【従来の方法の問題点】
各ショップのAPIを個別に利用する場合：
・楽天API → 楽天の商品のみ
・Amazon API → Amazonの商品のみ
・Yahoo! API → Yahoo!ショッピングの商品のみ
→ 3つのサイトしか検索できない

【SerpApiを使うメリット】
Google Shoppingには多数のショップが掲載されているため、
1回の検索で以下のサイトすべてを検索可能：

・Amazon
・楽天市場
・Yahoo!ショッピング
・ヨドバシカメラ
・ビックカメラ
・PayPayモール
・au PAY マーケット
・その他多数の通販サイト

→ 本当の「最安値」を見つけることができる

■ 運用コストの目安
━━━━━━━━━━━━━━━━━━━━━━━━

【小規模運用】
キーワード10個 × 1日1回実行
→ 月間約300検索
→ Freeプラン（無料）で対応可能

【中規模運用】
キーワード30個 × 1日1回実行
→ 月間約900検索
→ Freeプラン（無料）で対応可能

【本格運用】
キーワード50個 × 1日2回実行
→ 月間約3,000検索
→ Developerプラン（月額$50）


■ 検索結果の例
━━━━━━━━━━━━━━━━━━━━━━━━

キーワード「TOKIO INKARAMI シャンプー」で検索した場合：

1位: 楽天市場 - 3,980円
2位: Amazon - 4,200円
3位: Yahoo!ショッピング - 4,150円
4位: ヨドバシカメラ - 4,500円
5位: ビックカメラ - 4,480円

→ このように複数サイトの価格を一括で取得し、
  最安値から順に3件をスプレッドシートに出力します。


■ システムへの組み込み方
━━━━━━━━━━━━━━━━━━━━━━━━

1. SerpApiのアカウント作成（https://serpapi.com/）
2. APIキーを取得
3. システムの設定にAPIキーを登録
4. 自動リサーチ実行時にSerpApiで全サイト検索

※アカウント作成・APIキー取得の手順書はこちらでご用意いたします。

━━━━━━━━━━━━━━━━━━━━━━━━

■ まとめ
